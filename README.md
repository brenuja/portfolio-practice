# Portfolio Practice

A personal portfolio website showcasing projects, skills, and achievements. This project serves as a practice platform for building a responsive and visually appealing portfolio using modern web design principles.

## Features

- **Dynamic Portfolio Sections**:
  - About Me
  - Skills and Technologies
  - Projects with links and descriptions
  - Contact Form
- **Responsive Design**:
  - Optimized for mobile, tablet, and desktop screens.
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Hover effects on project cards and links

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/brenuja/portfolio-practice.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-practice
   ```
3. Open the `index.html` file in your browser to view the portfolio.

## Technologies Used

- **HTML**: Structures the portfolio content.
- **CSS**: Creates a visually appealing design.
- **JavaScript**: Adds interactivity and smooth transitions.

## Project Highlights

- **Modern UI/UX Design**:
  - Clean layout and easy-to-navigate structure.
- **Responsive Layout**:
  - Adapts to all screen sizes for a consistent experience.
- **Custom Styling**:
  - Personalized color schemes and typography.
